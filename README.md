# AimTrainer
Have fun aim training.

Train your aim for  fps games like Valorant, Counter strike, Call of Duty, Overwatch, Apex legends and many more.

Get better at all the available fps games by practicing your aim.

![alt text](https://github.com/Rohith2202/AimTrainer/blob/main/Aim%20Trainer-img.png?raw=true)


To run this game:

 1.Download the files or Clone the repository
 
 2.Open the Html file in the browser of your choice
 
 3.Play the game and imporve your aim
